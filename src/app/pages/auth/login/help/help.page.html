Welcome on Scaleway Manager!

Here is a list of questions/answers that might help you:

How to connect?

You need a Scaleway Elements account. Be careful, Scaleway Dedibox account doesn't work.

I'm having an issue, how to report it?

At the bottom of this page you have a mail icon, tap on it and write me what's wrong.

Is it an official application published by Scaleway?

Absolutely not! This is an Open Source Project on GitHub.

Is this app free and without ads?

This application is totally free and without any advertising and tracking.

Who created this app?

I'm only one developer, you can found me on Twitter: @matthpr_

What about our data/privacy?

I do not sell or use your data at any moment. I just use Scaleway API to allow you to connect, get your servers, start them, ect. Also this project is Open Source on GitHub.

If you're still worried about your account security I recommend you to activate Two-Factor Authentication.
